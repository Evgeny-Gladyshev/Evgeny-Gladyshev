## Телеметрия, датчики, Back-end, web-серверы

- На базе автономных небольших микропроцессоров с передачей данных через Wi-Fi/GSM/LTE сетей, или на базе радиоканала;
- ЧПУ станки, промышленные роботы и "умная оснастка";
- Создание систем управления на базе микроконтроллеров;
- Возможны как быстрые недорогие решения на базе No-code платформ, так и решения на основе промышленных стандартов;
- Программирование промышленных микроконтроллеров и создание различных комплексов на их базе;
- Чат-боты в соцсетях, с искуственным интеллектом;
- Чат-боты в соцсетях для взаимодействия с датчиками;
- Чат боты на сайте, с искуственным интеллектом и выводом данных в таблицы Google;
- Работа с базами данных;
